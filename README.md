> ʀᴀʀᴇᴘᴀɪʀs ɴᴀᴛᴜʀᴀʟ ᴇɴᴊᴏʏᴇʀ
>
> ɪɴ ᴀᴅᴅɪᴛɪᴏɴ ᴛᴏ ᴍʏ ᴅɴɪ, ᴍ/ᴡ (sᴛʀᴀɪɢʜᴛ sʜɪᴘs) ᴀɴᴛɪs ᴀɴᴅ ᴘʀᴏsʜɪᴘᴘᴇʀs. ʏ'ᴀʟʟ ᴄᴀɴɴᴏᴛ ʙᴇ sʀs ᴀʙᴏᴜᴛ ᴛʜɪs ɪ sᴡᴇᴀʀ.
